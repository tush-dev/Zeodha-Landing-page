# Zerodha Landing Page

A simple **Zerodha-inspired landing page** built using **HTML and CSS**.  
This project is made for practice to improve frontend development skills by recreating a clean and modern website layout.

## 🚀 Features

- Responsive landing page layout
- Clean navigation bar
- Hero section with branding
- Simple and modern UI
- Built only with HTML and CSS

## 🛠️ Tech Stack

- HTML5
- CSS3
